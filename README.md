# gllacy #
Выпускной проект HTML Academy "Профессиональный HTML и CSS, уровень 1". Магазин мороженного Глейси<br/>
Выполнил - **Паярели Александр**

## Описание проекта:
Веб-сайт "Cat Energy" состоит из двух страниц:
- [Главная страница](https://magistr19.github.io/gllacy "Ссылка на главную страницу")
- [Каталог продукции](https://magistr19.github.io/gllacy/catalog.html "Ссылка на каталог продукции")


Верстка выполнена резиново, веб-сайт изменяет свои размеры в пределах от 900 до 1200 пикселей.

### Использованные технологии:
- HTML 5
- CSS 3
- JavaScript
- Flexbox

### Особенности верстки:
- Резиновая, валидная, семантическая, кроссбраузерная верстка (IE10+)
- Оптимизированный вес сайта и его загрузка
- Сайт доступно отображается при незагруженных изображений/стилей
- Сайт доступен для плохо видящий людей, которые используют screen readers
- Сайт доступен с клавиатуры для людей с нарушенной моторикой, которым трудно пользоваться мышкой
